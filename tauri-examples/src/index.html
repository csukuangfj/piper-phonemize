Piper Phonemize - Tauri Example
GitHub

Description

  • Everything is open-sourced. code
  • piper-phonemize converts text to IPA phonemes using espeak-ng
  • Built with Tauri — native app with web frontend

About This Demo

  • Private and Secure: All processing is done locally on your device. No server is involved.
  • espeak-ng-data is embedded in the Rust binary (auto-extracted at runtime)
  • Supports macOS, Windows, Linux, and Android

Supported Languages

  • 110 languages supported including English, Chinese, Japanese, Korean, German, French, Spanish, Russian, Arabic, Hindi, and many more
